### Описание проекта
Угадать загаданнное компьютером число не более чем за 20 попыток

### Какой кейс решаем
Нужно написать программу, которая угадывает число не более чем за 20 попыток

**Условия соревнования:**
- Компьютер загадывает целое число от 0 до 100, и компьютер должен угадать за минимально количество попыток это число (не более 20 попыток)
- Алгоритм, который использует компьютер, основан на информации о том, больше ли случайное число или меньше нужного нам
- Для минимизации количества попыток в коде используем метод деления

**Метрика качества**
Результаты оцениваются по среднему количеству попыток при 100 повторений 

**Что практикуем**
Учимся писать хороший код на Python
